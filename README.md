## Usage
```bash
# nodejs version manager(Option)
$ npm install nvm -g
$ nvm install 10.15.0
$ nvm use 10.15.0
# npm resource manager(Option)
$ npm install nrm -g
$ nrm use taobao
# install, serve, build
$ npm install
$ npm run serve
$ npm run build
```
